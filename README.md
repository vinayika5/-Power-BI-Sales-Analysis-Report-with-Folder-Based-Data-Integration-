# 📊 Annual Sales Analysis Report (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI report analyzing annual sales performance across 9 states.

The report is built using a folder-based data integration approach, combining multiple state-level files into a single unified data model for comprehensive analysis.


## 🎯 Problem Statement

Businesses often manage data across multiple files or regions, making it difficult to analyze overall performance.

This project solves that problem by consolidating multiple datasets into a single report, enabling better insights into sales, profit, and regional trends.

---

## 🛠 Tools & Techniques Used

* Power BI
* Data Modeling
* Power Query (Folder Data Integration)
* Data Cleaning & Transformation
* DAX (for calculations)
* Interactive Report Design

---

## 🔄 Data Integration Approach

* Multiple state-level files were stored in a folder
* Data was imported using Power BI's **Folder connection**
* Files were combined and transformed using Power Query
* Created a unified dataset for analysis

---

## 📊 Report Features

### 🔹 Overview Page

* KPI Cards: Total Sales, Profit, Orders, Profit Margin
* Monthly Revenue & Profit Trend
* Category Contribution Analysis
* Geographic Map Visualization
* Revenue & Profit by Product

### 🔹 Detailed Analysis Page

* Sales Distribution Analysis
* Top 5 Sales Persons
* Top Performing Cities
* Product-wise Quantity Analysis
* Quarterly Performance Breakdown

### 🔹 Filters & Interactivity

* Month, State, City, Category, Product
* Sales Person slicers
* Reset slicer functionality

## 📈 Key Insights and Business recommendation

 Strong profitability (~20%)
→ Maintain pricing; push premium products to increase margins.
Revenue driven by Appliances & Electronics (~95%)
→ Focus marketing here; bundle Accessories to boost sales.
Sales concentrated in few states (Delhi, Karnataka, Gujarat)
→ Expand into weaker regions using targeted campaigns.
Top salespeople contribute disproportionately
→ Train others using top performers’ strategies + incentives.
Top cities dominate revenue (e.g., Howrah, Surat)
→ Invest in mid-tier cities for growth.
Few products drive most revenue (Laptop, AC, etc.)
→ Promote best-sellers; bundle or reduce low-performing products.Electronics and Appliances contribute the highest revenue
